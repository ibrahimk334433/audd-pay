# ⚡ AUDD Direct-Pay
### The Simplest Way to Accept AUDD on Solana

**AUDD Direct-Pay** is a lightweight, mobile-first payment link generator designed specifically for the **AUDD Stablecoin** ecosystem on Solana. It bridges the gap between freelancers and clients by replacing complex wallet addresses with intuitive, professional payment links.

## 🚀 The Problem
Sending long, complex Solana wallet addresses is prone to errors and creates friction for non-technical users, especially when dealing with the Australian Digital Dollar (AUDD) for real-world services.

## ✨ The Solution
Our tool allows merchants to generate a secure, professional checkout page in seconds.

* **Input:** Enter Solana wallet and AUDD amount.
* **Generate:** Get a unique URL to share.
* **Pay:** Clients open the link, connect Phantom, and pay instantly.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS (Mobile-First Design).
-   **Blockchain:** Solana Web3.js Integration.
-   **Hosting:** GitHub Pages (Live & Production Ready).

## 📱 Features
* **Auto-Redirection:** Seamless flow from link generation to checkout.
* **Wallet Detection:** Identifies Phantom Wallet and supports Deep Linking for mobile users.
* **Dynamic Parameters:** Uses URL parameters to pass payment data securely.
